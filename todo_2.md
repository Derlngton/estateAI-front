
Фаза 1
Пропала иконка мессенджера с карточек агентов. Исходя из поля messenger, полученного с бекенда, нужно доббавлять подходящую иконку из папки src/assets/icons


Фаза 2
Мне не нравится элемент, расположенный за статусом агента. Там какой-то кружок. Текст вылезает из него. Замени кружок на что-то более адаптивное под меняющийся текст статуса. Так же присвой этой фигуре свой цвет, исходя из статуса
Возможныые статусы(не изменяй регистры): 'Working','Stopped','Done','Banned'


Фаза 3. При нажатии на созданную карточку должна открываться отдельная заполненная форма со всей инфой по агенту.Для получения одного конкретного агента нужно использовать эндпоинт гет http://localhost:8000/agents/get_agent/{agent_id} без тела. Все поля формы должны быть редактируемыми при нажатии на них. Если что-то было отредактировано, должна стать активной кнопка "Сохранить изменения". На этой форме так же должны быть кнопки "Назад", "Запустить агента", "Удалить агента"

Фаза 4. Кнопка "Сохранить изменения". При нажатии отправляет PUT запрос к http://localhost:8000/agents/{agent_id}
с телом запроса:                    
    "name"
    "phone"
    "avatar"
    "messenger"
    "type"
    "description"
даже если было изменено только одно поле, нужно отправить все

Фаза 5. Кнопка удаления агента. При нажатии отправляет Delete запрос к http://localhost:8000/agents/{agent_id}
